# API 키 입력 안내

API 키를 생성하셨다면, 다음 방법 중 하나로 설정하세요:

## 방법 1: .env 파일 직접 수정 (가장 간단)

1. `.env` 파일을 엽니다 (방금 생성되었습니다)
2. `your-api-key-here` 부분을 실제 API 키로 교체하세요
3. 저장하세요

예시:
```
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## 방법 2: 배치 파일 사용

```cmd
간단_키설정.bat
```

실행 후 API 키를 입력하세요.

## 방법 3: Python 스크립트 사용

```bash
python quick_login.py
```

또는

```bash
python set_key_direct.py
```

## 확인

설정 후 테스트:
```bash
python claude_client.py
```

## 주의사항

- API 키는 `sk-ant-`로 시작합니다
- `.env` 파일은 Git에 커밋되지 않습니다 (안전합니다)
- API 키를 다른 사람과 공유하지 마세요
