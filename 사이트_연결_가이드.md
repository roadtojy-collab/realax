# Anthropic 콘솔 연결 가이드

## 🔗 직접 링크

**Anthropic 콘솔:** https://console.anthropic.com/

## 📋 API 키 생성 방법

1. **콘솔 접속**
   - 위 링크를 클릭하거나 브라우저에 주소 입력

2. **로그인**
   - 계정이 있으면 로그인
   - 계정이 없으면 "Sign Up"으로 계정 생성

3. **API 키 생성**
   - 왼쪽 메뉴에서 "API Keys" 클릭
   - "Create Key" 버튼 클릭
   - 키 이름 입력 (선택사항)
   - 생성된 API 키를 복사 ⚠️ **한 번만 표시되므로 반드시 복사하세요!**

4. **API 키 설정**
   - 복사한 키를 사용하여 로그인:
     ```bash
     python quick_login.py
     ```
   - 또는 배치 파일 사용:
     ```cmd
     간단_키설정.bat
     ```

## 🚀 빠른 실행

터미널에서:
```bash
start https://console.anthropic.com/
```

또는 Python 스크립트:
```bash
python open_console.py
```

## 📝 참고

- API 키는 `sk-ant-`로 시작합니다
- API 키는 한 번만 표시되므로 안전한 곳에 보관하세요
- 키가 유출되면 즉시 콘솔에서 삭제하세요
