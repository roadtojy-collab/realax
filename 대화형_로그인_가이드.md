# 대화형 로그인 가이드

## 현재 상태 확인

### 1. Python 설치 확인
터미널에서 실행:
```bash
python --version
```
또는
```bash
py --version
```

### 2. 필요한 파일 확인
다음 파일들이 있는지 확인:
- ✓ login.py
- ✓ claude_client.py
- ✓ requirements.txt

## 대화형 로그인 실행

### 방법 1: 직접 실행 (권장)
```bash
python login.py
```

### 방법 2: 확인 후 실행
```bash
python start_login.py
```

## 로그인 프로세스

1. **스크립트 실행**
   ```bash
   python login.py
   ```

2. **API 키 선택**
   - 옵션 1: 이미 API 키가 있는 경우 → 직접 입력
   - 옵션 2: API 키가 없는 경우 → Anthropic 콘솔에서 생성

3. **API 키 입력**
   - API 키를 입력하세요 (일반적으로 `sk-ant-`로 시작)

4. **저장 방법 선택**
   - 옵션 1: 현재 세션에만 저장
   - 옵션 2: .env 파일에 저장 (영구, 권장)
   - 옵션 3: 둘 다 저장

5. **연결 테스트**
   - 자동으로 연결 테스트를 진행합니다

## 문제 해결

### Python이 실행되지 않는 경우
1. Python이 설치되어 있는지 확인
2. 새 터미널 창 열기
3. Python 경로가 PATH에 포함되어 있는지 확인

### 파일이 없는 경우
모든 파일이 프로젝트 디렉토리에 있는지 확인하세요.

## 수동 로그인 (대안)

Python이 실행되지 않는 경우, 직접 .env 파일을 생성할 수 있습니다:

1. `.env` 파일 생성
2. 다음 내용 입력:
   ```
   ANTHROPIC_API_KEY=your-api-key-here
   ```
3. `python-dotenv` 설치:
   ```bash
   pip install python-dotenv
   ```
